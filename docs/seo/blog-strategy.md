# Kodinav Blog Strategy

Goal: rank for commercial-intent and problem-aware searches that lead businesses
to hire a web developer. Every topic below maps to a real search intent and links
naturally to a service page (internal-link target in brackets).

Publishing rhythm: 2–4 posts/month. Depth beats frequency — 1,200+ words,
one search intent per post, written in the studio's engineer-to-business-owner voice.

## Categories

1. **Website Development** — buying-stage content for people comparing options
2. **Performance & SEO** — technical authority content (the studio's edge)
3. **E-Commerce** — store owners researching platforms and conversion
4. **Business Growth** — how software decisions affect revenue
5. **Local (Delhi NCR)** — city-intent content for local rankings
6. **Technology Explained** — React/Next.js/stack content for credibility
7. **Industry Playbooks** — coaching institutes, clinics, real estate, hotels
